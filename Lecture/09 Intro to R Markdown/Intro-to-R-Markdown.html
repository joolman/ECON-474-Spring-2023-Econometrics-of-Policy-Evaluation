2 Markdown Basics

Markdown is an easy-to-read text formatting language that converts plain text into a pretty document. R Markdown is Rs version of Markdown that can incorporate, execute, and output R code in the final knitted document. To knit the document type Ctrl_Shift-K

2.1 Writing Text

This is how to write in Markdown. Notice writing on the next line of code stays in the same paragraph.

This is how to start a new paragraph.

Wrap your words in * to get italics. Using two *s will produce boldface. You can combine _ in any order to get boldfaced italics.

2.3 LaTeX Math Integration

To write math, we can use some limited \(\LaTeX\) features. Like if we wanted to show a regression formula inline: \(y_i = \beta_0 + \beta_1 x_i + \epsilon_i\). Alternatively, if we have some important math, we can make it stand out: \[ 2 + 2 = 2\times 2 = 2^2 = 4 \approx \pi \] OR, if we really want to show off knowing the geometric series identity, we can show it by aligning it with &: \[ \begin{align} \sum_{n=0}^\infty ar^n &= a \left( \frac{1}{1-r} \right) \tag{1.1} \\ & = \frac{a}{1-r}\\ & \text{for } |r| < 1 \end{align} \]

2.6 Writing Code

Here is the basics to writing code chunks.

print('hello world')
## [1] "hello world"

The first part of the curly braces tells you what language to use and the name of the code chunk. Then there are options, which can be set up after a comma and setting them to either TRUE or FALSE. Some of the major options are:

  • include - Whether or not to be included in the document
  • echo - Display the code
  • results - Displays the output of the code
  • messages - Displays messages from the functions (if they have them)
